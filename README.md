# Walentynka

A web project built with Vite and Vanilla TypeScript.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is configured to be deployed on GitHub Pages at `https://jamadamur.github.io/walentynka/`.
